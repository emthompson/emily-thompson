NBM Show

Event registration system

While at NBM I designed a sucessful event registration system that supported multiple events throughout the country. The new service integrated with the exsiting user management system and event set up was controlled via a CMS. During the event the browser-based process ran on the check-in kiosks to allow attendees to check in and print badges. Once implemented, NBM saw record numbers of online registrations and high user satisfaction.
